Creating Custom Skins for R5Reloaded and R5Valkyrie

Overview

This page details the process on creating custom skins and importing them into Modded Apex Legends, for whichever project you wish to use, whether it is R5Valkyrie or R5Reloaded,
if you wish to have your skins

Requirements

  • 3D modeling software (e.g., Blender, Maya)
  • Texture editing software (e.g., Photoshop, GIMP)
  • Basic understanding of rigs and animations
  • R5 Reloaded installed

Porting Steps

Step 1: Export your model from your software as FBX or OBJ.
Step 2: Check normals, UVs, and materials.
Step 3: Convert textures to DDS and adjust compression.
Step 4: Import model and textures into R5 Reloaded.
Step 5: Test your asset in-game and verify animations. 
// Example dummy command
r5r_import --model "dummy_model.fbx" --textures "textures/" --output "r5r_assets/"

Common Issues

  • Textures not showing: Check file paths and formats.
  • Animations not aligned: Verify skeleton structure.
  • Scaling issues: Confirm model units match game units.
  • Shader problems: Some materials may need manual adjustment.
